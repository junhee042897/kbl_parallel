# KBL Parallel Coordinates Starter

## 실행 방법
1. 이 폴더를 VS Code로 엽니다.
2. `index.html` 파일을 엽니다.
3. Live Server 확장 기능으로 실행합니다.

## 포함 파일
- index.html
- style.css
- script.js
- kbl.csv

## 현재 구현 기능
- 시즌 필터
- 승패 필터
- 팀 필터
- win / lose 색상 구분
- 대표 지표 기반 평행좌표계 렌더링
